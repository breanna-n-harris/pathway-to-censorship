# pathway-to-censorship
Supplemental material for “The Death of Academic Freedom at a Nationally Ranked University: A warning to the academic community.”

This folder contains pdfs of the documents and policies relevant to the pathway to censorship at Texas Tech University. The pdfs are here as a reference to verbiage/versions present at the time of submission. We did not author the pdfs contained in this folder. All of this information is freely available and is part of the public record. Original authors hold copyright.
